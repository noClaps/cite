# Cite

A tool to generate [Hayagriva](https://github.com/typst/hayagriva) citations from DOI URLs.

## Build instructions

You'll need [Monopack](https://codeberg.org/noClaps/monopack).

Clone the repository.

```bash
git clone https://codeberg.org/noClaps/cite.git
```

Build the site.

```bash
monopack src/index.html
```

This will place the built files in the `dist/` directory.
